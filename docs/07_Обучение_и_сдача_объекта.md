## 9.1 Программа обучения персонала охраны (ЧОП / дежурный)

**Продолжительность:** 4 часа (теория 1 ч + практика 3 ч)  
**Место проведения:** административное здание, рабочее место оператора  
**Участники:** ___________ (ФИО, должность)

### Блок 1. Основы работы с системой (1 час)

- [ ] Включение / выключение монитора и рабочего места оператора
- [ ] Вход в систему под учётной записью оператора
- [ ] Интерфейс живого просмотра: раскладки, масштаб, PTZ (при наличии)
- [ ] Переключение между камерами, полноэкранный режим
- [ ] Что означают индикаторы онлайн/офлайн у камер

### Блок 2. Работа с архивом (1 час)

- [ ] Поиск записи по дате и времени
- [ ] Поиск по событию (движение, тревога, LPR)
- [ ] Воспроизведение: пауза, ускорение, перемотка
- [ ] Экспорт фрагмента видео на USB-носитель
- [ ] Как запросить просмотр архива для жителя (процедура)

### Блок 3. Работа с LPR (1 час)

- [ ] Открытие журнала въезд/выезд
- [ ] Поиск по номеру автомобиля
- [ ] Поиск по периоду (например, «кто въезжал вчера с 10:00 до 12:00»)
- [ ] Экспорт журнала в Excel
- [ ] Как добавить/удалить номер в белый или чёрный список
- [ ] Как работает оповещение о нераспознанном номере

### Блок 4. Тревожные события и аварии (30 мин)

- [ ] Что делать при тревоге датчика шума (сигнал, квитирование)
- [ ] Что делать если камера показывает «офлайн»
- [ ] Как сообщить в техническую поддержку (контакт, порядок)
- [ ] Действия при отключении электричества (UPS, сколько работает система)

### Блок 5. Практика (30 мин)

- [ ] Самостоятельный поиск события в архиве — оператор справился: ✓/✗
- [ ] Самостоятельный поиск по номеру в LPR — оператор справился: ✓/✗
- [ ] Экспорт видео на флешку — оператор справился: ✓/✗

## 9.2 Краткая инструкция оператора (памятка)

> *Распечатать и разместить на рабочем месте охраны*

### Ежедневные действия оператора

**Начало смены:**
1. Войти в систему (логин: `chop_operator`, пароль: см. конверт у руководства)
2. Проверить: все ли камеры онлайн? Если есть офлайн — записать в журнал и сообщить ответственному
3. Проверить: архив пишется? (Индикатор записи в правом углу)

**В течение смены:**
- Следить за монитором в режиме live
- При срабатывании тревоги шума — проверить камеру площадки, при необходимости выйти
- При въезде незнакомого автомобиля — проверить в журнале LPR

**Конец смены:**
- Передать пост, сообщить о любых инцидентах

### Как найти видео по времени
1. Нажать «Архив» → выбрать камеру → выбрать дату → выбрать время → Play

### Как найти по номеру машины
1. «LPR» → «Журнал» → ввести номер → нажать поиск
