## О проекте

Настоящий пакет документов описывает проектирование, поставку и монтаж системы видеонаблюдения для коттеджного посёлка «Аркадия» (140 участков). Система охватывает все ключевые зоны посёлка: КПП с распознаванием номеров, 6 перекрёстков, детскую площадку, административное здание и насосную станцию.

### Ключевые параметры

| Параметр | Значение |
|---|---|
| Протяжённость дорог | ~3,5 км |
| Количество камер | 20 (вариант A) / 28 (вариант B) |
| Распознавание номеров (LPR) | 2 камеры на КПП (въезд + выезд) |
| Идентификация лиц | 1 камера (вар. A) / 9 камер (вар. B) |
| Цветная ночная съёмка | На всех обзорных камерах (ColorVu) |
| Архив хранения | 30 календарных дней |
| Датчик шума | 1 шт. (без записи речи) |
| Столбов освещения | 93 шт. (с закладными для монтажа) |
| Кабельных колодцев | 38 шт. (с оптическими муфтами) |
| Удалённый доступ | VPN (WireGuard), мобильное приложение |
| Резервное питание | UPS 1500 ВА (центр) + АКБ на каждом узле |

### Два варианта комплектации

**Вариант A (минимальный):** 2 LPR + 17 обзорных + 1 ID-камера — покрытие всех обязательных зон с идентификацией лиц на пешеходном входе.

**Вариант B (рекомендованный):** 2 LPR + 17 обзорных + 9 ID-камер — полное покрытие с идентификацией лиц на всех перекрёстках, площадке и насосной.

### Два сценария сетевой инфраструктуры

**Сценарий 1 — оптика провайдера:** быстрый старт, минимальные затраты на инфраструктуру, ежемесячная аренда канала.

**Сценарий 2 — собственная ВОЛС:** полная независимость, кольцевая топология с резервированием, без ежемесячных платежей.
