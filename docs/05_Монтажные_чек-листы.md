- [ ] Отрыта траншея 0,8 м по трассе

- [ ] Кабель уложен в траншею / протянут через ПНД-трубы
- [ ] Засыпка траншеи с сигнальной лентой «ОПТИКА» на глубине 0,3 м
- [ ] Восстановлено покрытие (асфальт/брусчатка) в местах вскрытия
- [ ] Замерена длина кабеля по факту: _____ м

### 4.5 Монтаж муфт МТОК-96А (в каждом узле)

**Узел №___ / Перекрёсток _______________**

- [ ] Волокна сварены (сплайсер + OTDR замер: потери ≤0,1 дБ на сварку)
- [ ] Запиговано: волокно → патч-корд SC/APC → в шкаф к SFP-порту коммутатора
- [ ] Муфта закрыта, опрессована, установлена в шкафу/колодце
- [ ] OTDR-рефлектограмма сохранена (файл: `OTDR_Узел_N_дата.sor`)

### 4.6 OTDR-измерения всего кольца

- [ ] Измерение выполнено с двух концов кольца
- [ ] Затухание на каждом участке не превышает бюджет линии

- [ ] Файлы сохранены: `ВОЛС_Аркадия_OTDR_дата/`

# ЭТАП 5. Монтаж PoE-коммутаторов и ИБП в шкафах ВРУ

### 5.1 Для каждого шкафа ВРУ (Узлы 1–8)

**Узел №___ / Адрес/Описание _______________**

**Подготовка шкафа:**
- [ ] Шкаф ВРУ исправен, дверь закрывается на ключ
- [ ] Внутри шкафа есть DIN-рейка / место для монтажа оборудования (≥3U)
- [ ] Имеется розетка 220V или клемма питания внутри шкафа
- [ ] Шкаф оснащён термостатом (обогрев/вентиляция)

**Монтаж оборудования:**
- [ ] Dahua PFS4210-8GT-DP-V2 установлен на DIN-рейку / закреплён
- [ ] Аккумулятор 12В/17Ач установлен, зафиксирован
- [ ] Зарядный модуль PS-BOX установлен, подключён к аккумулятору и коммутатору
- [ ] Провода подключены, клеммы затянуты, защита от КЗ установлена
- [ ] Маркировка всех кабелей выполнена (кабель UTP — номер камеры; оптика — номер волокна)
- [ ] Коммутатор включён, индикация питания — ОК

**Настройка коммутатора:**
- [ ] IP-адрес управления установлен согласно таблице адресации: 10.200.0.1X
- [ ] Имя устройства задано: `SW-UzelN`
- [ ] VLAN 100 (камеры) настроен на портах 1–8 (access)
- [ ] VLAN 200 (управление) настроен на uplink-портах SFP
- [ ] Пароль администратора изменён, сохранён в парольный менеджер проекта
- [ ] RSTP/STP включён (для кольцевой топологии — ERPS)
- [ ] Тест PoE: подключена тестовая камера — питание и ping ОК

## 6.1 Общий порядок работ

1. Разметка точки монтажа на столбе по согласованной схеме
2. Установка кронштейна / хомута
3. Протяжка кабеля UTP через закладную столба
4. Установка камеры на кронштейн
5. Разводка кабеля от основания столба до шкафа (в гофрошланге по земле или по кабель-каналу)
6. Подключение кабеля к PoE-порту коммутатора в шкафу

# ЭТАП 7. Установка центрального узла (Административное здание)

## 7.1 Монтаж серверного шкафа

- [ ] Место под шкаф ШН-9U подготовлено (розетка 220V + заземление рядом)
- [ ] Шкаф установлен, закреплён к стене
- [ ] UPS APC 1500 ВА установлен в шкаф (1U) или рядом
- [ ] NVR/сервер установлен в шкаф (1–2U)
- [ ] Агрегационный коммутатор CSS4024 установлен в шкаф (1U)
- [ ] Патч-панель установлена, кабели промаркированы
- [ ] Органайзеры кабельные установлены

## 7.2 Коммутация

- [ ] SFP-кабели / патч-корды от агрегационного коммутатора к NVR подключены
- [ ] Оптические патч-корды от всех 8 узлов подключены к агрегационному коммутатору (SFP-порты)
- [ ] HDD установлены в NVR (4 или 6 дисков согласно варианту)
- [ ] NVR подключён к UPS
- [ ] Монитор оператора подключён к NVR (HDMI)
- [ ] UPS протестирован (имитация отключения питания, NVR продолжает работу)

## 7.3 Проверка связности центрального узла

- [ ] Ping до каждого коммутатора (Узлы 1–8): ✓/✗
- [ ] NVR видит все камеры (все онлайн): Вариант A — 21 шт., Вариант B — 29 шт.
- [ ] Скорость записи соответствует расчётной (вкладка состояния NVR)
- [ ] Запись ведётся на все камеры по расписанию (24/7)
- [ ] Архив доступен за последние сутки: ✓/✗
